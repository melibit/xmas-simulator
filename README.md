# xmas-simulator server
- INSTALL QEMU (https://www.qemu.org/download/).
- INSTALL THE ARDUINO-CLI (https://docs.arduino.cc/arduino-cli/installation/)
- MAKE SURE THESE ARE IN YOUR %PATH%
```
python3 emu_server.py
```
